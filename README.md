# 🎮 Juego del Número Secreto - JS Game

¡Bienvenido al proyecto del **Juego del Número Secreto**! Esta es una aplicación interactiva desarrollada con JavaScript vainilla, HTML5 y CSS3, donde el usuario debe desafiar a la lógica intentando adivinar un número generado aleatoriamente por el sistema.

https://lacroux.github.io/juego-numero-secreto/

---

## 🚀 Características

- **Generación Aleatoria:** El juego selecciona un número secreto en cada nueva partida.
- **Validación de Intentos:** El sistema indica si el número secreto es mayor o menor al ingresado por el usuario.
- **Contador de Intentos:** Al ganar, el juego muestra cuántas oportunidades necesitaste.
- **Interfaz Responsiva:** Diseño moderno con fondos degradados y efectos visuales de "ruido" y "código".
- **Accesibilidad:** Uso de fuentes legibles (Chakra Petch e Inter) y elementos semánticos.

---

## 🛠️ Tecnologías utilizadas

* **HTML5:** Estructura del juego y contenedores.
* **CSS3:** Diseño personalizado con Flexbox, gradientes lineales y efectos de superposición de imágenes.
* **JavaScript:** Lógica del juego, manipulación del DOM y manejo de eventos.
* **Google Fonts:** Fuentes personalizadas para una estética moderna y tecnológica.

---

## 📐 Estructura del Proyecto

El repositorio contiene la base para el funcionamiento del juego:

* `index.html`: La estructura principal del juego.
* `style.css`: Estilos visuales, incluyendo el layout responsivo y los estados de los botones.
* `app.js`: (Debe incluirse) La lógica que controla las funciones `verificarIntento()` y `reiniciarJuego()`.

---

## 🕹️ Cómo jugar

1.  Clona este repositorio o descarga los archivos.
2.  Abre el archivo `index.html` en tu navegador favorito.
3.  Ingresa un número en el campo de texto (Input).
4.  Haz clic en el botón **"Intentar"** para validar tu respuesta.
5.  Si aciertas, se habilitará el botón **"Nuevo juego"** para comenzar una nueva partida.

---

## 🎨 Personalización Visual

El CSS incluye detalles avanzados como:
- **Fondo:** Un gradiente profundo desde azul hasta negro.
- **Efecto de Código:** Una imagen de fondo con opacidad al `0.4` que simula un entorno de programación.
- **Botones Dinámicos:** Cambios de color cuando el botón está deshabilitado (`disabled`).

---

## 🤝 Contribuciones

Este proyecto es ideal para practicar lógica de programación. Si deseas mejorarlo, puedes:
1.  Aumentar el rango del número secreto (ej. del 1 al 100).
2.  Añadir un límite máximo de intentos.
3.  Incluir sonidos al acertar o fallar.
